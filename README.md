# Dagster Pipeline + Great Expectations Quality Gates

This repository now includes the `dagster-gx-pipeline` project, a complete local data pipeline showcasing Dagster orchestration, Great Expectations data quality gates, and DuckDB storage.

## Quick start

```bash
cd dagster-gx-pipeline
python -m pip install -e .[dev]
make dev
```
